Etiquetas HTML Comunes

Una guía breve sobre las etiquetas básicas de HTML

1. Etiqueta de Encabezado (<h1> - <h6>)

Las etiquetas de encabezado, de <h1> a <h6>, definen los niveles de encabezados en la página. <h1> es el encabezado más importante, mientras que <h6> es el menos importante. Ejemplos:

primer heading

segundo heading

tercer heading

cuarto heading

quinto heading
sexto heading

2. Etiqueta de Párrafo (<p>)

La etiqueta <p> se usa para definir párrafos de texto en la página. Permite separar bloques de contenido para que el texto sea más fácil de leer. Este mismo parrafo es el ejemplo

3. Etiqueta de Enlace (<a>)

La etiqueta <a> crea enlaces a otras páginas o secciones de la misma página. Usualmente se combina con el atributo href para definir la URL del enlace.

Ejemplo: Pagina principal

4. Etiqueta de Imagen (<img>)

La etiqueta <img> se usa para insertar imágenes en una página. Requiere al menos el atributo src para indicar la URL de la imagen y el atributo alt para describirla.

5. Etiquetas de Listas (<ul>, <ol>, <li>)

Las listas no ordenadas <ul> y ordenadas <ol> se utilizan para crear listas de elementos. Cada elemento de la lista se define con <li>.

Ejemplo de lista no ordenada:

                
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
