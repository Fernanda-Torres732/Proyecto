Introducción a CSS

Guía básica sobre cómo estilizar elementos en HTML utilizando CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de documentos HTML. Nos permite controlar el color, las fuentes, los márgenes, el espaciado, la distribución y muchos otros aspectos visuales de los elementos en una página web.

2. Sintaxis de CSS

La sintaxis de CSS consta de selectores y declaraciones. Un selector apunta al elemento HTML que queremos estilizar, y una declaración especifica el estilo que aplicaremos a dicho elemento.

Ejemplo de una regla CSS:

selector {
    propiedad: valor;
}

Por ejemplo, para cambiar el color del texto de todos los párrafos:

p {
    color: blue;
}

3. Formas de aplicar CSS

Existen tres formas principales de aplicar CSS:

  • CSS en línea: usando el atributo style dentro de una etiqueta HTML. Ejemplo: <p style="color: red;">Texto rojo</p>
  • CSS interno: en la sección <style> del documento HTML.
  • CSS externo: en un archivo separado con extensión .css que luego enlazamos en el HTML.

4. Selectores Básicos de CSS

Los selectores de CSS nos permiten seleccionar y estilizar elementos específicos en HTML. Algunos selectores básicos son:

  • Selector de etiqueta: selecciona todos los elementos de una etiqueta específica. Ejemplo: p { color: blue; }
  • Selector de clase: selecciona elementos con una clase específica, usando el símbolo .. Ejemplo: .miClase { font-size: 20px; }
  • Selector de ID: selecciona un elemento con un ID específico, usando el símbolo #. Ejemplo: #miID { background-color: yellow; }

5. Propiedades Comunes de CSS

Algunas propiedades de CSS comúnmente utilizadas incluyen:

  • color: establece el color del texto. Ejemplo: color: red;
  • background-color: define el color de fondo. Ejemplo: background-color: blue;
  • font-size: ajusta el tamaño del texto. Ejemplo: font-size: 18px;
  • margin: controla el espacio exterior de un elemento. Ejemplo: margin: 10px;
  • padding: controla el espacio interior de un elemento. Ejemplo: padding: 10px;
