¿Qué es HTML?

Definición de HTML

HTML, o HyperText Markup Language, es el lenguaje estándar para crear y estructurar el contenido en la web. Utiliza "etiquetas" para definir los elementos que se mostrarán en la página, como texto, imágenes, enlaces y más.Es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.

¿Para qué sirve HTML?

HTML permite estructurar y organizar el contenido en una página web. Con HTML, puedes crear:

  • Encabezados y párrafos
  • Listas
  • Imágenes y enlaces
  • Tablas y formularios

Ejemplo básico de HTML

Un documento HTML básico tiene una estructura simple que comienza con la declaración de tipo de documento y varias etiquetas principales. Ejemplo:

                
<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <title>Página de Ejemplo</title>
    </head>
    <body>
        <h1>¡Hola, mundo!</h1>
        <p>Esta es una página web sencilla.</p>
    </body>
</html>
