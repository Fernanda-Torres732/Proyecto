Formularios en HTML

Una guía rápida sobre los elementos básicos de los formularios en HTML

1. Etiqueta <form>

La etiqueta <form> es el contenedor principal de un formulario en HTML. Utiliza atributos como action para especificar la URL donde se enviarán los datos y method para definir el método de envío, como POST o GET.

Ejemplo: <form action="procesar.php" method="POST">...</form>

2. Campos de Entrada (<input>)

La etiqueta <input> se usa para capturar diferentes tipos de datos de los usuarios. El tipo de dato se define con el atributo type, que puede ser text, email, password, etc.

Ejemplo: <input type="text" name="nombre" placeholder="Ingresa tu nombre">

3. Área de Texto (<textarea>)

La etiqueta <textarea> se utiliza para campos de texto largos, como comentarios o descripciones. A diferencia de <input>, tiene etiquetas de apertura y cierre.

Ejemplo: <textarea name="comentarios" rows="4" cols="50"></textarea>

4. Lista Desplegable (<select> y <option>)

La etiqueta <select> crea una lista desplegable, y cada opción se define con la etiqueta <option>.

Ejemplo:

                
<select name="pais">
    <option value="mx">México</option>
    <option value="us">Estados Unidos</option>
</select>

5. Botón de Envío (<button> o <input type="submit">)

El botón de envío se utiliza para enviar los datos del formulario. Puedes crearlo con la etiqueta <button> o con <input type="submit">.

Ejemplo: <button type="submit">Enviar</button>

6. Captura de Datos con JavaScript

Con JavaScript, podemos capturar y mostrar los valores ingresados por el usuario en un formulario. Utilizamos el método document.getElementById() para obtener un elemento por su ID y el método document.querySelector() para seleccionar elementos basados en selectores CSS.

Ejemplo:

var nombre = document.getElementById('nombre').value;
document.getElementById('mostrarNombre').textContent = nombre;
var edad = document.querySelector('input[name="edad"]:checked').value;

En este código:

  • querySelector('input[name="edad"]:checked') selecciona la opción marcada dentro de un grupo de radio buttons con el nombre edad, capturando así el valor de la edad seleccionada.
