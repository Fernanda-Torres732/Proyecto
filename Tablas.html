Tablas en HTML

Una introducción básica a las tablas en HTML y sus etiquetas principales

1. Etiqueta <table>

La etiqueta <table> es el contenedor principal para crear una tabla en HTML. Dentro de esta etiqueta, se colocan las filas y celdas de la tabla.

Ejemplo básico: <table>...</table>

2. Filas de la Tabla (<tr>)

La etiqueta <tr> se usa para definir una fila en la tabla. Cada fila contiene una serie de celdas de datos o encabezados.

Ejemplo: <tr>...</tr>

3. Encabezado de Columna (<th>)

La etiqueta <th> se utiliza dentro de una fila (<tr>) para definir una celda de encabezado de columna. Estas celdas suelen aparecer en negrita para indicar que son títulos de cada columna.

Ejemplo: <th>Nombre</th>

4. Celdas de Datos (<td>)

La etiqueta <td> define una celda de datos dentro de una fila (<tr>). Esta etiqueta se usa para cada dato que se quiera mostrar en la tabla.

Ejemplo: <td>Juan</td>
